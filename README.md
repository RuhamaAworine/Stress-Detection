# Stress-Detection
## Overview
This project predicts stress level (Low, Medium, High) based on lifestyle factors using Machine Learning.
## Features
- Sleep hours
- Heart rate
- Study hours
- Screen time
## Model
Random Forest Classifier
## Tech Stack
Python, Pandas, Scikit-learn
## How to Run
1. Install requirements
2. Run notebook
3. Load model.pkl and test prediction

## Result
Achieved good accuracy using basic ML pipeline
